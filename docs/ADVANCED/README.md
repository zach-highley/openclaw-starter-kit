# Advanced Playbook

This folder contains the “extra” layer: the patterns that turn a basic OpenClaw setup into a **high-autonomy, high-reliability system**.

## Contents
- `TRUST_LADDER.md` — 5-level autonomy policy system
- `TOKEN_BURN_STRATEGY.md` — how to pace and parallelize serious work
- `OVERNIGHT_BUILD_PIPELINE.md` — queue → execution → morning summary
- `DAILY_RHYTHM.md` — daily planner + review loop
- `DASHBOARD_INTEGRATION_PATTERNS.md` — dashboard patterns + safe integration
- `EMAIL_LABEL_MIGRATE_MECE_RUNBOOK.md` — MECE email labels + migration patterns
- `EMAIL_IMAP_APPLE_MAIL.md` — making labels visible and usable in Apple Mail
- `IOS_APP_BUILD_WORKFLOW.md` — how we use Codex for iOS app work
- `FELIXCRAFT_PATTERNS.md` — inspiration + actionable patterns

## Safety
Everything here is written to be public:
- no secrets
- no hardcoded tokens
- no personal identifiers

If you copy/paste scripts, configure via env vars.
